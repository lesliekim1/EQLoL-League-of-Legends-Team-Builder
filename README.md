# EQLoL-League-of-Legends-Team-Builder

EQLoL: League of Legends Team Builder is a team builder web application for the game [League of Legends](https://www.leagueoflegends.com/en-us/). It helps users to explore champions, builds, and items to build and share a team. EQLoL is designed for League of Legends players of any rank and skill who are already familiar with the game.
